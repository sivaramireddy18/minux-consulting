### 2. Microscopic Daily Blueprint

#### 📘 Theory Deep-Dive (4 Hours)
Understand how the standard allocator manages memory. Learn about allocation requests (`malloc`, `calloc`), block resizing (`realloc`), and memory recycling (`free`). Comprehend fragmentation, memory leaks, and double-free exceptions.

#### 🛠️ Unassisted Lab Track (4 Hours)
1. Write a program that intentionally leaks memory and track it with Valgrind.
  2. Observe stack trace errors generated by double-frees or invalid memory accesses.
