### 2. Microscopic Daily Blueprint

#### 📘 Theory Deep-Dive (4 Hours)
Understand what happens inside the preprocessor. Study header file inclusions (`#include`), conditional compilation (`#ifdef`, `#ifndef`, `#else`, `#endif`), header guards to prevent multiple definition errors, and macro definitions (object-like and function-like macros).

#### 🛠️ Unassisted Lab Track (4 Hours)
1. Write a C file with multiple nested headers and inspect the preprocessed output file (`.i`) generated by compiling with `-E`.
  2. Implement robust macro guards and note the elimination of duplicate declarations.
