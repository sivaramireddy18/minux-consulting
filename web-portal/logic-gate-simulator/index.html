01. Theoretical Foundations

NAND/NOR Universality

Any boolean function can be realized using solely NAND or NOR gates. In silicon, CMOS NAND gates are preferred as they use N-channel transistors in series (faster mobility than P-channel series in NOR).

Propagation Delay ($t_{pd}$)

The time taken for an input transition to propagate and reflect at the output. In physical circuits, gate delays limit the maximum operating frequency ($f_{max} = 1 / t_{propagation}$).

Thermal Noise Margins

The measure of noise immunity of a gate. Calculated as: $NM_H = V_{OH} - V_{IH}$ and $NM_L = V_{IL} - V_{OL}$.

02. Reference Circuits

1-Bit Full Adder

Computes sum ($S$) and carry-out ($C_{out}$) from bits $A$, $B$, and carry-in ($C_{in}$):

S = A ⊕ B ⊕ Cin
Cout = (A & B) | (Cin & (A ⊕ B))

SR Latch (Feedback Loop)

A bistable multivibrator utilizing feedback loops. Realized by connecting the output of one NOR gate to the input of the second NOR gate, forming a 1-bit static memory cell.

04. Industrial Use Cases

ALU Overflow Flags

In CPU ALUs, the Overflow flag ($V$) indicates signed arithmetic overflow. It is generated using a single XOR gate monitoring the carry-in and carry-out of the most significant bit ($V = C_{in} \oplus C_{out}$).

Address Decoders

Logic decoder circuits route CPU address bus lines to activate specific memory blocks (e.g. select line for SRAM vs FLASH) using combinational AND/NAND grids.
