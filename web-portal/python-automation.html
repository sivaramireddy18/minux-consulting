HIL Test Automation Architects

Vetting specification for automation consultants ready to build multi-threaded test loggers, write SCPI instrumentation drivers, configure pytest scopes, and lock down CI/CD pipelines.

Verification & Vetting Milestones

Phase 1

Embedded Python & Hardware Scripting

Verified competencies: Implement host-side scripts wrapping toolchains and flash tools, write unbuffered PySerial byte readers, unpack raw binary packets with CRC-16 checks, and program MicroPython peripherals.

Weekly Breakdown

  • Module A: Subprocesses stream capturing, paths pathlib portability & argparse CLIs
  • Module B: Serial communications (PySerial), struct binary packaging & CRC-16 checks
  • Module C: MicroPython, register pins peripheral controls & memory-safe interrupt ISRs
Phase 2

Test Automation & Hardware-in-the-Loop (HIL)

Verified competencies: Program lab instrument connection wrappers (SCPI) using PyVISA resource manager context blocks, build stateless pytest fixtures, and design automated relay fault-injections.

Weekly Breakdown

  • Module A: Instrumentation wrappers, PyVISA resource mapping & SCPI syntax trees
  • Module B: Pytest structures, yield setup/teardown fixtures & parameterized sweeps
  • Module C: HIL closed loops, relay control & power brownout/short-circuit fault injection
Phase 3

Advanced Automation & Binary Auditing

Verified competencies: Parse compiled firmware ELF segments (.text, .data, .bss) and symbol address offsets in Python, build multi-threaded data loggers, and compile Click CLI test report builders.

Weekly Breakdown

  • Module A: ELF layouts parsing (pyelftools), memory segment sizing & pre-flashing gates
  • Module B: Concurrency thread limits (GIL), thread-safe queues & CSV/Parquet data-logging
  • Module C: Nested CLI commands (Click), Matplotlib graphing & Weasyprint PDF reports
Phase 4

Enterprise Framework & CI/CD

Verified competencies: Write type-safe scripts under strict Mypy checks, apply connection Singleton factory patterns, write automated Jenkinsfile pipelines, parse JUnit XML metrics, and build HIL test matrices.

Weekly Breakdown

  • Module A: Mypy static types, Singleton managers, Factory loaders & Black/Ruff check automations
  • Module B: CI/CD continuous integration (GitHub Actions), JUnit XML parser & Slack webhooks
  • Module C: Enterprise HIL framework, concurrent exceptions handling & emergency de-energize recovery
